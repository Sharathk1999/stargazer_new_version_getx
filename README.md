# Stargazer

## Key Features

- AudioPlayer and Asset Audio Player Integration
- Favorite, Recent, and Most Played Tracks
- Mini Player 
- Local Data Storage with Hive 
- Audio Controls:
  - Next, Previous
  - Fast Forward, Skip
  - Repeat, Shuffle
  - Progress bar, Stop, Pause, etc.
- Playlist Management

### State Management
